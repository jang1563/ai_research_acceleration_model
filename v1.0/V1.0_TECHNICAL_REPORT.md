### 2.2 Mathematical Framework

**Acceleration Calculation:**

```
A(domain, year) = A_base × T(year) × S(domain, year) × M(scenario)
```

Where:
- `A_base`: Domain base acceleration (calibrated from literature)
- `T(year)`: Time evolution factor (1 + 0.08 × (year - 2024))
- `S(domain, year)`: Cross-domain spillover boost
- `M(scenario)`: Scenario modifier (0.6 - 1.8)

**Spillover Calculation:**

```
S(target) = Σ log(1 + A_source - 1) × coefficient × lag_factor × 0.3
```

Capped at 50% per source to prevent unrealistic compounding.

**Uncertainty:**

```
CI_90 = [A - width, A + width]
where width = A × uncertainty × (1 + 0.05 × t)
```

Uncertainty grows with time horizon.

### 3.1 Main Class

### 3.2 Core Methods

| Method | Returns | Description |
|--------|---------|-------------|

## 7. Usage Examples

### Quick Start

```python
from ai_acceleration_model import AIAccelerationModel

model = AIAccelerationModel()
print(model.executive_summary(2030))
```

### Detailed Analysis

```python
# Compare scenarios
scenarios = model.compare_scenarios("drug_discovery", 2030)
for name, forecast in scenarios.items():
    print(f"{name}: {forecast.acceleration:.1f}x [{forecast.ci_90[0]:.1f}-{forecast.ci_90[1]:.1f}]")

# System trajectory
trajectory = model.trajectory(None, 2025, 2035)
for snapshot in trajectory:
    print(f"{snapshot.year}: {snapshot.total_acceleration:.1f}x, {snapshot.workforce_change:+.2f}M jobs")

# Policy recommendations
for rec in model.get_policy_recommendations(2030)[:3]:
    print(f"[{rec.priority.upper()}] {rec.title}")
```

## 8. Limitations

1. **Parameter uncertainty**: Base parameters from literature review, not direct measurement
2. **Scenario probabilities**: Expert judgment, not empirically derived
3. **Spillover coefficients**: Estimated from case study analysis
4. **Workforce projections**: High uncertainty in job creation rates
5. **Regulatory evolution**: Political factors difficult to model
6. **International context**: Primarily US/EU perspective
